# Service Bulletin: Li-ion Battery Safety

| Symbol | Hazard | Precaution |
| :---: | :--- | :--- |
| ![Electric](images/electric.png) | **High Voltage** | Disconnect battery before servicing controller. |
| ![Charging](images/charging.png) | **Fire Hazard** | Use only the original 42V/2A charger. |
| ![No Water](images/no-water.png) | **Chemical Fire** | Do NOT use water on a battery fire. Use Class D powder. |

## 1. Charging Procedure
1. Inspect the Julet connector for debris.
2. Plug the charger into the wall first, then the battery.
3. Stop charging if the casing exceeds 45°C. ![Hot](images/hot.png)
